# Avatsi Website

Static GitHub Pages website for Avatsi, a home automation and audio video solution company.

## Structure

- `index.html` - homepage with hero, services, featured spaces, and CTA
- `about.html` - company overview and process
- `contact.html` - contact details and project inquiry CTA
- `assets/styles.css` - shared site styling
- `assets/main.js` - mobile navigation toggle and footer year
- `assets/images/` - local site imagery, including downloaded legacy photos from the old Wix site

## GitHub Pages

This site is plain HTML, CSS, JavaScript, and SVG, so it can be deployed directly with GitHub Pages without a build step.

## Legacy Resources

Selected images were downloaded from the earlier Wix website and stored in `assets/images/legacy/` for use in the redesign.
